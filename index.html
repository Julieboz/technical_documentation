Introduction

The Flexible Box Module, known as Flexbox, is one of two modern layout systems. As a one-dimensional layout model, it controls the layout based on a row or a column.

The main goal of flex box is to allow items fill the whole space of their container, based on rules you set.

Enable Flexbox

To apply the flexbox layout to a container, set the following

display: flex;

or

display: inline-flex;

Placing this CSS property on an element allows you to use other flex properties. The content inside the container will be aligned using flexbox.

To set general rules for the container’s items, the following flexbox properties can be applied:

  • flex-direction
  • justify-content
  • align-items
  • flex-wrap
  • flex-flow
Flex Direction

The flex-direction property determines if the container should align its items as a row or columns as follows:

  • flex-direction: row places items as a row, in the text direction left to right
  • flex-direction:row-reverse places items like row but it the opposite direction
  • flex-direction:column places items as a column ordering top to bottom
  • flex-direction:column-reverse places items like column but in opposite direction

By default, items start from the left if flex-direction is set to row, and the top if flex-direction is set to column.

Justify Content

The justify-content property tell the CSS how to align and space out flex items along it’s main axis, or the direction in which they are arranged. There are 6 common values for this property:

  • center: aligns all flex items to the center inside the flex container; this is the most common property
  • flex-start: aligns items to the start of the container. In a row, pushes items to the left; in a columns, pushes items to the top. **This is the default alignment if no justify-content value is selected.
  • flex-end: aligns items to the end of the container.
  • space-between: aligns items to center of main axis, with extra space placed between items. The first and last items are pushed to the edge of the flex container.
  • space-around: similar to space-between but first and last items are not locked to the edge of the container. Space is distributed all around the items with 1/2 space on either end of the container.
  • space-evenly: Distributes space evenly between the flex container items with a full space at either end of the container.
Align Items

The align-items property works similar to justify-content except it aligns flex items along the cross axis rather than the main axis. For rows, it tells how to push items up or down; for columns, how to push left to right.

There are 5 values for this property:

  • flex-start: align to the start of the container (top or left)
  • flex-end: align to the end of the container (bottom or right)
  • center: align to the vertical center of the container
  • stretch: items are stretched to fit the container. *Default value if no value is specified.
  • baseline: align to the baseline of the container
Flex Wrap

By default, items in a flexbox container are kept on a single line, shrinking them to fit the container. To force items to spread across multiple lines, is the flex-wrap property. By setting this property, extra items move into a new row or column.

The break point of where wrapping happens depends on the size of the items or the size of the container.

The following values can be applied:

  • nowrap: default value, does not wrap items
  • wrap: wraps items from left to right if in rows, or top to bottom if in columns
  • wrap-reverse: reverses direction - either right to left or bottom to top
